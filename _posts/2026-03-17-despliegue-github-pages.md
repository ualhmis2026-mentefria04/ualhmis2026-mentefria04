# Despliegue de nuestro blog en la nube

**Fecha**: 17 de marzo de 2026
**Autor**: Marcos Escánez

Como parte de los requisitos de nuestro proyecto de la asignatura HMIS, hemos logrado publicar nuestro sitio estático de forma pública. 

El primer despliegue lo hemos realizado utilizando **GitHub Pages**, una herramienta fantástica que se integra directamente con nuestro repositorio y nos permite ver los resultados de forma rápida.

## Proceso seguido para el despliegue
Para lograrlo, hemos seguido la metodología de trabajo colaborativo basada en ramas:
1. Configuración del archivo `_config.yml` para asentar las bases de Jekyll.
2. Creación de la estructura principal del blog, incluyendo la carpeta `_posts` para las noticias.
3. Activación de GitHub Pages desde los ajustes (*Settings*) del repositorio, configurando el despliegue automático desde nuestra rama `main`.

¡El flujo de trabajo con *GitHub Flow* nos está resultando muy útil para coordinar el trabajo del equipo sin pisarnos el código!
