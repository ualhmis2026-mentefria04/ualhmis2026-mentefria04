Sobre nuestro equipo

Somos el equipo Mentefria04, formado por Marcos Escánez Clemente y Martín de Vicente Márquez.

En este proyecto estamos desarrollando de forma colaborativa un blog estático utilizando Jekyll y automatizando su despliegue tanto en GitHub Pages como en la nube de Azure. Nuestro objetivo es dominar el flujo de trabajo con Git y GitHub Flow, además de aprender a gestionar repositorios en equipo.
